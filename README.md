# Fourth-year-Applied-mathemtics-codes

> A collection of the practical work, assignments, and projects completed during my fourth year as an Applied Mathematics Engineering student at the National Higher School of Mathematics (NHSM), Algeria.

## About

This repository contains the Python codes I developed throughout different modules during my fourth year of engineering studies.

The objective of this repository is to keep all my implementations in one place while documenting my learning journey in image processing and computer vision... ,  Most of the codes were written during laboratory sessions, coursework, and personal practice to better understand the concepts taught in class.

## Repository Structure

```
.
├── MS_Codes_Python/      # Python implementations of course concepts
├── images/               # Sample images used for testing

```

## Getting Started

Clone the repository

```bash
git clone https://github.com/Oussama-zeroual/Fourth_Year_Applied_Mathematics_Codes.git
```

Move into the project

```bash
cd Fourth_Year_Applied_Mathematics_Codes
```

Install the required libraries

```bash
pip install opencv-python numpy matplotlib scikit-image
```

Run any Python script

```bash
python script_name.py
```
